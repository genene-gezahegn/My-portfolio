# 🚀 START HERE - Genene Gezahegn Portfolio

Welcome! This is your premium portfolio website. Follow these steps to get started.

## ⚡ Quick Start (5 Minutes)

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```

### 3. Open in Browser
Visit: http://localhost:3000

### 4. Customize Your Content
Edit these files with your information:
- `src/components/Hero.jsx` - Your name and title
- `src/components/About.jsx` - Your story
- `src/components/Projects.jsx` - Your projects
- `src/components/Contact.jsx` - Your contact info

### 5. Deploy to Vercel
```bash
npm i -g vercel
vercel
```

## 🆘 Need Help?

### Common Issues

**Port already in use?**
```bash
npx kill-port 3000
```

**Build failing?**
```bash
rm -rf node_modules package-lock.json
npm install
```

**Animations not working?**
- Clear browser cache
- Check console for errors
